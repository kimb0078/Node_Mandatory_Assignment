Introduction

Node.js er en JavaScript runtime, som bruger Chrome's V8 Engine til at køre JavaScript direkte på desktop computeren i stedet for i browseren.

Chrome og Node.js giver os custom libraries med metoder/funktioner, som ikke findes i JavaScript dokumentationen. Da disse funktioner ikke er en del af JavaScript, kalder de C++ metoder i V8 engine, som laver alt arbejdet bag scenen.
